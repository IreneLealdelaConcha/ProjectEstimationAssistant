Project Estimation Assistant

Interactive chatbot to estimate project timelines

This Copilot assists with early-stage time estimates for automation projects using Robotic Process Automation (RPA), Conversational AI or Low-code. It collects key project parameters, calculates a rough effort estimate, and outputs a total hours with buffer. Built in Microsoft Copilot Studio for the AI Chatbot Developer assessment.
